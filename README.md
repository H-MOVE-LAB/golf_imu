# IMU Error Propagation in Position Reconstruction and Its Mitigation through Task-Informed Dual IMU Constrained Optimization: Application to Golf Putting

## Overview

This repository contains code and data related to the paper:

> **IMU Error Propagation in Position Reconstruction and Its Mitigation through Task-Informed Dual IMU Constrained Optimization: Application to Golf Putting**

The project investigates error propagation in inertial measurement unit IMU-based position reconstruction and proposes a task-informed constrained optimization framework using dual IMUs to mitigate drift and improve trajectory estimation accuracy in golf putting applications.
